Timestamp.js

Just the timestamp

timestamp() // the Timestamp object

Format

// Custom formats
timestamp().format("MM DD, yyyy h:ii:ss x") // 
timestamp().format("WW") // 
timestamp().format("M DD, yy") // 
timestamp().format() // 
// Built-in formats
timestamp().format("short") // 
timestamp().format("24hour") // 
timestamp().format("full") // 
timestamp().format("12hour") // 
// A format error
timestamp().format("m/d/y") // 
timestamp().format("fail") //  (Work in progress)

timestamp().day() //

timestamp().minute() //

timestamp().meridiem() //  (Not super useful alone, but it's there)

Start/End of

timestamp().startOf("month").format("MM DD, yyyy h:ii:ss x")

timestamp().endOf("month").format("MM DD, yyyy h:ii:ss x")
